
Movie Trailer Website : is a static web page allowing vistors to browse their movies and watch the trailers.

How to open it -:
	
	1- install python
	2- search on your computer about a programm called "IDLE python GUI" 
	3- from top left corner there is a menu called file open it and the second one called "open"
	4- open a file caled "entertainment_center.py"
	5- from menu bar you will find "Run" menu click on it or press "f5" 
	6- the project will be opened in your browser 
	7- you can browse movies and watch their trailer by a click
